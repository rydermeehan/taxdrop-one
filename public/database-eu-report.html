Methodology

This Equal & Uniform analysis (Texas Tax Code §41.43(b)(3)) compares the subject property to a representative sample of similar properties within the same Central Appraisal District. Comparables are selected on (1) same property class, (2) similar living area (60–150% of subject sqft), (3) similar year built (within ±15 years where available), and (4) proximate geography (same ZIP code or subdivision preferred, county-wide fallback). The indicated value equals the median per-square-foot value of the selected comparables × the subject's living area. Per §41.43(b)(3), if the appraisal district's value exceeds the median appraised value of a reasonable number of comparable properties appropriately adjusted, the protest must be granted.

All comparable property data was extracted directly from the Central Appraisal District's official 2026 record. Source CAD records are publicly available and verifiable.
